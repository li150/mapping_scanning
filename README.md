## 一、关于产品

### 产品简介

- [配图扫描] 能够高效的提取出扫描照片文本内容
- [配图扫描] 将纸质文档扫描成电子文本，便于存储、高亮匹配
- [配图扫描] 能将识别文字后对选择选中的文本进行百度百科
- [配图扫描] 在无需网络连接的情况下，依然能够完成文字识别操作，保障用户隐私和安全
